## Latest Releases

### v0.19.1 (January 10, 2026)

**Bug Fixes:**
- Fixed MyHeritage GEDCOM whitespace-only line handling
- Added async GEDCOM parsing for large files (800KB+) to prevent UI freezing
- Improved anonymization script

### v0.19.0 (January 9, 2026)

**Major Update: Plugin Rename**
- Plugin renamed from "Canvas Roots" to "Charted Roots"
- Automatic vault migration for existing users
- Backward compatibility for old protocol handlers and command IDs

### v0.18.32 (January 9, 2026)

**Improvements:**
- Automatic wikilink resolution to `cr_id` values
- Wikidata place web clipper template

### v0.18.31 (January 8, 2026)

**New Features:**
- GEDCOM anonymization tool for safe issue reporting
- Enhanced staging promotion workflow
- PersonIndexService integration improvements
